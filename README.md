# Frutiger Aero Blog

A modern, minimalist blog inspired by the Frutiger Aero aesthetic - a design style characterized by clean, futuristic interfaces with glassmorphism effects, vibrant gradients, and a digital-age look.

## Features

- **Glassmorphism Design**: Translucent cards with blur effects
- **Vibrant Gradients**: Characteristic Frutiger Aero color schemes
- **Responsive Layout**: Works on all device sizes
- **Interactive Elements**: Hover effects and animations
- **Modern Typography**: Clean, readable fonts

## Design Elements

- Frosted glass UI elements
- Bright, saturated color gradients
- Subtle animations and transitions
- Clean typography with good spacing
- Minimalist aesthetic with functional beauty

## How to Use

1. Open `index.html` in your web browser
2. The site will automatically apply the Frutiger Aero styling
3. All CSS and JavaScript files are included in the project

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter)
- JavaScript (for interactive elements)

## Browser Support

For the best experience, use a modern browser that supports:
- CSS backdrop-filter
- CSS Grid and Flexbox
- CSS Variables
- Modern JavaScript (ES6+)
